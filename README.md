# 🏭 設備稼働率ダッシュボード

PythonとStreamlitを使用して作成した設備稼働率可視化アプリです。

GitHub Actionsによる定期データ生成と、Streamlit CloudによるWeb公開を行っています。

## 🌐 アプリURL

https://equipment-operation-dashboard-midkcrlr9bksdimu9be5dd.streamlit.app/

---

## 📋 概要

製造業では設備の停止状況や稼働率を把握し、
改善活動へ繋げることが重要です。

本アプリでは設備ごとの稼働率を集計し、
日別・設備別の推移を可視化できるようにしました。

---

## 🚀 主な機能

### 設備別稼働率表示

- 設備ごとの実績稼働率表示
- 想定稼働率との比較

### 月別推移グラフ

- 日別稼働率推移
- 設備ごとの比較表示
- 実績：実線
- 想定：点線

### 停止情報管理

- 停止時間
- 停止理由
- 生産数
- 不良数

### 自動データ生成

GitHub Actionsにより

- 0:00
- 6:00
- 12:00
- 18:00

に自動実行されます。

## ⚙ GitHub Actions

設備稼働率データはGitHub Actionsにより定期生成されています。

休業日判定

- 日曜日
- 日本の祝日

はデータ生成をスキップします。

また休み明けの始業時間を考慮し、
計画時間を自動調整しています。

## 👨‍💻 開発背景

現在、製造業向けDX業務に携わっており、

- 設備稼働率
- 歩留まり
- 生産実績

などの可視化業務を行っています。

業務理解を深めるため、
個人開発として設備稼働率ダッシュボードを作成しました。

DBを使用したかったですが、Pythonで接続情報を書かないといけないため、
個人情報の漏洩を考慮し、CSV転記に変更しております。
